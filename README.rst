NS Collector
==============

Script for gathering data out of NS.

Install
=======

With python2.7 and pip > 9.0 ::

  sudo pip install ~/path/to/NSCollector

Run
===

See: ::

  ns-collector -h

Notes:

Typical test run ::

  sudo pip uninstall --yes nscollector; sudo pip install ~/NSCollector/; ns-collector -h
